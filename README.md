# FullMoon

A lightweight LLM chat application built with Rust and Iced.

## Features

- Chat with various LLMs through OpenRouter
- Tree like chat structure
- Clean and simple interface
- Fast and native performance

## Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/fullmoon.git
cd fullmoon
```

2. Set up your development environment:

**Option A: Using Nix**
```bash
nix develop
```

**Option B: Manual setup**
- Install Rust from [rust-lang.org](https://rust-lang.org)
- Install Iced framework dependencies from [iced](https://github.com/iced-rs/iced/blob/master/DEPENDENCIES.md)

## Running the Application

```bash
cargo run
```

## Configuration

To use FullMoon, you'll need to set up your OpenRouter API key in the app settings.
